# MCU-Purple-Baseline# MCU Purple Baseline

Purple policy server for **MCU AgentBeats (MineStudio)**, implemented using the **A2A (Agent-to-Agent) protocol**.

This repository provides a minimal but fully functional Purple Agent that:
- Runs an A2A-compliant HTTP server
- Receives `init` / `obs` messages
- Responds with `ack` / `action` payloads
- Exposes an Agent Card for discovery by Green agents and evaluators

---

## Requirements

- Python **>= 3.11**
- OS: Windows / Linux / WSL
- Internet connection (for dependency installation)

---

## Installation

### 1. Create and activate virtual environment

```bash
python -m venv .venv
